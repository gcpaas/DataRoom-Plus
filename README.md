🔥DataRoom-Plus是一款基于SpringBoot3.x、Vue3.x、Element-plus、G2Plot、Echarts等技术栈的大屏设计器，具备大屏设计、预览能力，支持MySQL、Oracle、PostgreSQL、SQLServer、ElasticSearch、JSON、HTTP、Groovy等数据集接入，使用简单，完全免费，代码开源。<br/>

## 特别说明
功能未开发完成, 请勿用于生产环境
